# 🚀 Smart Economato - Presentación de Requisitos

Este repositorio contiene la plataforma interactiva diseñada para la **exposición final** del proyecto **Smart Economato**. 

En esta web se detallan todos los requerimientos funcionales, las fases de desarrollo y la arquitectura técnica implementada durante el ciclo formativo.

## 📋 Contenido de la Exposición

La plataforma está dividida en varias secciones clave:

*   **Página Principal:** Introducción, experiencia personal y cronograma de trabajo (Diagrama de Gantt interactivo).
*   **Informes Técnicos:**
    *   **Primeras Fases:** Definición del problema, investigación y decisiones iniciales.
    *   **Fase de Desarrollo:** Arquitectura hexagonal, patrones de diseño y seguridad.
    *   **Viabilidad V.1:** Características implementadas (escáner de productos, trazabilidad inmutable, chat IA).
    *   **Accesibilidad (WCAG 2.1):** Evaluación de cumplimiento y mejoras de inclusión.
    *   **Estudio UX/UI:** Heurísticas de Nielsen, perfiles de usuario y experiencia en tiempo real.
*   **Stack Tecnológico:** Detalle de todas las tecnologías utilizadas en el ecosistema (Angular 21, Java 25, Kafka, NestJS, etc.).

## 🛠️ Tecnologías utilizadas en esta presentación

Para esta plataforma de exposición se han utilizado:

*   **HTML5 / CSS3 (Vanilla):** Diseño premium, responsive y con soporte nativo para modo oscuro/claro.
*   **JavaScript:** Lógica de navegación fluida y sincronización de componentes.
*   **Mermaid.js:** Renderizado de diagramas de Gantt y flujos de trabajo.
*   **Google Fonts:** Tipografía Inter para una estética moderna y profesional.

## 🚀 Cómo visualizarlo

Simplemente abre el archivo principal en cualquier navegador moderno:

1.  Clona el repositorio.
2.  Abre el archivo `index.html`.
